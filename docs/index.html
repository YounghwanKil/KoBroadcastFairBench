KoBroadcastFairBench

A benchmark project for context-aware fairness evaluation in Korean broadcast scripts.

Korean broadcast scripts toxic / non-toxic fairness taxonomy GPT/Codex silver-label labeling SWA baseline
Status: benchmark construction in progress. This public page contains schemas, taxonomy, prompts, metadata, and pilot labels only. It does not redistribute raw AI Hub or broadcaster text/audio.

Goal

Build a real Korean broadcast-script fairness benchmark, create GPT/Codex silver-label toxic/non-toxic and taxonomy silver labels, freeze a GPT/Codex silver-label test set, and evaluate whether RoBERTa + SWA outperforms non-SWA baselines.

Real broadcast scripts→ Codex silver labels→ GPT/Codex silver-label locking→ Locked benchmark→ SWA ablation
